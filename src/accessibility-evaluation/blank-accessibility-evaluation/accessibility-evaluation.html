Please read the instructions on how to conduct this accessibility evaluation before proceeding with the evaluation.

  1. Make sure Node is installed on your computer.
  2. On Windows, open the "File Explorer" application
  3. Locate the folder that contains this HTML file.
  4. On Windows, right click on the folder to show the context menu and select the option called "Open in Terminal". On a Mac, select the option called "{{WHAT}}". A new application window will open.
  5. Type node server in the application window.
  6. Copy the following URL {{WHAT}} and paste it in the address bar of your Internet browser. Press enter.
  7. Start the accessibility evaluation. You can edit the evaluation right on the Internet browser. On Windows, you can press "ctrl" + "s" to save your progress.
